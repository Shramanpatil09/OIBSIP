# Quiz App

## 📱 Description
A simple Android Quiz Application developed using Java and XML in Android Studio.

The app displays multiple-choice questions to users, allows answer selection, and calculates the final score at the end of the quiz.

---

## ⚙️ Features
- Multiple-choice quiz questions
- Interactive UI
- Score calculation
- Instant result display
- User-friendly design

---

## 🛠️ Technologies Used
- Java
- XML
- Android Studio
