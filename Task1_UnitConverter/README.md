# Unit Converter App

## 📱 Description
An Android application that converts units like:
- Meter ↔ Kilometer
- Gram ↔ Kilogram

## ⚙️ Features
- Easy unit conversion
- Clean UI
- Fast calculations

## 🛠️ Technologies Used
- Java
- XML
- Android Studio
