# Todo App

## 📱 Description
A simple Android Todo Application developed using Java, XML, and SQLite in Android Studio.

The application allows users to:
- Register and login
- Add daily tasks
- View task list
- Delete completed tasks

All task data is stored locally using SQLite database.

---

## ⚙️ Features
- User Registration
- User Login Authentication
- Add Tasks
- Delete Tasks
- SQLite Database Integration
- Simple and Clean UI

---

## 🛠️ Technologies Used
- Java
- XML
- SQLite
- Android Studio
