# 🚀 Oasis Infobyte Internship Projects (OIBSIP)

This repository contains Android application projects developed during the AICTE Oasis Infobyte Internship Program using Java and XML in Android Studio.

---

# 📱 Projects Included

## 🔹 Task 1 — Unit Converter App
A simple Android application for converting units like:
- Meter ↔ Kilometer
- Gram ↔ Kilogram

### Features
- Fast calculations
- Simple UI
- Easy conversion system

📂 Folder: `Task1_UnitConverter`

---

## 🔹 Task 2 — Quiz App
An Android Quiz Application with multiple-choice questions and score calculation.

### Features
- Interactive quiz interface
- Multiple-choice questions
- Final score display

📂 Folder: `Task2_QuizApp`

---

## 🔹 Task 3 — Todo App
A complete Todo management application with SQLite database support.

### Features
- User Login & Registration
- Add/Delete Tasks
- SQLite Database
- Daily task management

📂 Folder: `Task3_TodoApp`
